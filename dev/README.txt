-- How to make a server on your PC:--
Open this link and download FABRIC Installer
-> https://fabricmc.net/use

-- Click the latest Fabric Installer and choose 1.18.1 and install server 0.12.12 into a new folder > Download server jar > Press Generate > Done > Exit
> Drag mods Folder, Configs Folder, start.bat, server-icon and server.properties into the new folder you made 
> run the fabric-server-launch (If this comes up with an error it means you need Java 17)
> accept EULA agreement 
> launch the start.bat file

Java 17 Required!
https://adoptium.net/?variant=openjdk17
